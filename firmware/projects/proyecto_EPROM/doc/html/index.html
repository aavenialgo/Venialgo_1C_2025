Proyecto EPROM

General Description

Este es el proyecto integrador de la asignatura electronica programable. El mismo consiste la utilizacion de dos sensores, uno de ppg, montado en una placa y otro de biopotenciales (para levantar un ECG), los cuales se encuentran conectados a un microcontrolador ESP32. El objetivo del proyecto es la lectura de los datos de ambos sensores y su posterior envio a ECG a una script de Pyhton mediante bluetooth y PPG a un puerto serie.
